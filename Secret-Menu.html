Secret Menus

Food Branding / Web Development / Web design

Secret menus is a website where you can find special menus from select restaurants located in Vancouver, BC. This website is built on an MVC framework, you can add restaurants on the main page and add secret menu items with full detail such as their image, price, rating, and name of each item under the restaurant page. You can effortlessly search for a restaurant with the search box on the main page or find the restaurant by browsing through the category section at the top of the page. Also, the website is associated with Amazon as well, so customers can see what others are purchasing and what products are must haves. However, a challenging aspect of this website was creating the PHP code for uploading the pictures. It is necessary that the customers can upload the images of the items, so it was needed to create this function, but there were a few difficulties while doing it.
